
NikeShop is a **Sample** shoe store application that work with Rest Api.

**In This App I use:**

- Koin Dependency Injection
- MVVM Architecture
- Kotlin
- Retrofit
- Room
- RxJava & RxAndroid
- Event Bus
- Navigation Component
- Fresco
